## 四、算法路线

### 4.1 全局环路由（已废弃）

可行域边界向内 buffer 生成 15 条等距轮廓环 ring 0~14。  
**致命缺陷**：ring 14 弧长 3.7mm，位于板中央，密集簇深层 pad 分配后 stub 飞出板框（oob=42）。

### 4.2 Option-A 局部环 ✅（当前主方案）

**核心**：对每个簇的 `cluster_poly = unary_union(forbidden_zones)` **向外** buffer：

```python
local_ring_i = cluster_poly.buffer(HALF_BUF + i*PITCH)
             .exterior ∩ feasible_area  →  ConformalRing
```

**排序**：按距簇外边界距离升序（外圈 pad → ring 0，内圈 → 更深）。

**注入点**：环上 300点均匀采样，找最近可直线到达的点。

**结果**：oob=**0** ✓，dirty=53（C2/C3 2D密集，几何固有限制，不影响保序验证）。

**脚本**：`pcb-mesh-routing/option_a_local_rings.py`

### 4.3 A* 网格路由 ✅（已验证）

```python
routing_poly     = board - other_nets_forbidden_zones
wire_center_poly = routing_poly.buffer(-LINE_W/2)
# 网格 1944×276 ≈ 536K 格，A* 耗时 0.31s
```

String-pulling（贪婪二分）：1955格点 → 21折点。

**验证**：I2S1_MOSI_SYS_AR1（x=85.7↔225.7），148.56mm，in_board=True ✓

**脚本**：`pcb-mesh-routing/astar_grid_router.py`

## 七、学术对标

本项目局部等距轮廓线槽 = **PLC (Pin-Edge-Chain-Channel) 模型**：

> 马博宇等，《面向柔性电路板的通道区域自动布线算法》  
> 计算机辅助设计与图形学学报，2022  
> https://www.jcad.cn/cn/article/pdf/preview/10.3724/SP.J.1089.2022.19465.pdf

商业对应：Zuken「Contour Routing」、Cadence Allegro「Contour & Arc Routing」。

流程定位：
```
Global Routing
  → Channel Planning（局部环分配）
  → Track Assignment（depth 排序）
  → Detailed Routing（A*）
```
